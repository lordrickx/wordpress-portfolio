## 🚀 Installation & Usage

### Prerequisites
- WordPress 5.0+
- PHP 7.4+
- Node.js 14+
- ACF Pro (for ACF components)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/lordrickx/wordpress-portfolio.git
cd wordpress-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Build assets**
```bash
npm run dev
```

4. **For production**
```bash
npm run production
```
